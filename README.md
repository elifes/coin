What is eLifescoin?
----------------

eLifescoin is a lite version of Bitcoin using scrypt as a proof-of-work algorithm.

1 - 49999 block :
   1 - 100000 ELSK Reward

50000 - 149999 block :
   1 - 10000 ELSK Reward

150000 - 199999 block :
   1 - 1000000 ELSK Reward

200000 - 399999 block :
   500 ELSK Reward

400000 - 599999 block :
   2500 ELSK Reward

600000 block + :
   25 + Transaction Fees Reward
